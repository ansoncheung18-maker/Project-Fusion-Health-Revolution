# 月球太空電梯專案 Phase A：概念設計與可行性論證

**專案名稱：Project Lunar Skyhook**  
**版本：2.0**  
**更新日期：2026-06-03**  
**核心目標：建造「短纜索 + 火箭」混合運輸系統，大幅降低月球運輸成本。**

---

## 1. 願景與目標

> 傳統太空電梯需要 60,000 公里纜索同未量產嘅碳納米管。本方案採用 **「短纜索 + 火箭」混合模式**，用現有材料即可建造，技術可行、成本可控。

### 1.1 核心目標

- 建造一條長度 **2,000 公里** 嘅旋轉纜索（太空天鉤 Skyhook）
- 配合傳統火箭（如 Starship），將地球至月球嘅運輸成本降低 **50-70%**
- 年運量：**10,000-20,000 噸**
- 總投資：**$50-100 億美元**

---

## 2. 點解「短纜索 + 火箭」？

| 對比項 | 傳統太空電梯（月球） | **短纜索 + 火箭混合** |
|:---|:---|:---|
| 纜索長度 | 60,000 km | **2,000 km** |
| 材料要求 | 碳納米管（未量產） | **Zylon / 玻璃纖維（已有）** |
| 纜索質量 | 725 萬噸 | **約 1,000 噸** |
| 技術成熟度 | 極低 | **中等（可今日建造）** |
| 建造成本 | $800-900 億 | **$50-100 億** |
| 每公斤運費（地 → 月） | ~$100 | **~$500-1,000** |

---

## 3. 技術架構

### 3.1 系統組成

| 組件 | 位置 | 功能 |
|:---|:---|:---|
| **旋轉纜索（天鉤）** | 低月球軌道（約 100 km 高） | 旋轉產生離心力，捕捉/釋放飛船 |
| **月球錨點** | 月球表面 | 固定纜索一端（可選） |
| **升降機** | 沿纜索移動 | 月球表面 ↔ 天鉤 |
| **傳統火箭** | 地球 → 月球軌道 | 將貨物送至天鉤交匯點 |

### 3.2 運作流程
地球發射（Starship）
↓
與天鉤交匯（月球軌道）
↓
貨物轉移至天鉤
↓
天鉤旋轉將貨物「甩」向月球表面
↓
或：升降機沿短纜索下降至月球表面

### 3.3 纜索設計

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 長度 | 2,000 km | |
| 材料 | Zylon PBO | 抗拉強度 5.8 GPa |
| 直徑 | 10 cm | |
| 質量 | 約 1,000 噸 | |
| 旋轉速度 | 約 1 km/s | 產生離心力 |
