# 月球太空電梯專案 Phase B：子系統設計 (Subsystem Design)

**版本：1.0**  
**更新日期：2026-06-03**

---

## 1. 纜索子系統

### 1.1 設計目標
- 承受 60,000 公里自身重量 + 升降機載重
- 抵抗微隕石撞擊
- 使用現有商業化材料

### 1.2 技術方案

| 組件 | 規格 | 說明 |
|:---|:---|:---|
| 材料 | Zylon PBO | 抗拉強度 5.8 GPa |
| 直徑 | 100 mm | 安全係數 3.9 |
| 結構 | 多股編織 | 1,000 條幼線，斷一條唔影響 |
| 表面處理 | 抗紫外線塗層 | 抵抗太陽輻射 |

### 1.3 纜索製造

| 方法 | 說明 |
|:---|:---|
| 太空製造 | 在 L1 點太空站「生長」Zylon 纖維 |
| 分階段運送 | 先運 1,000 噸設備到月球 |
| 編織 | 將幼纖維編織成 100 mm 纜索 |

## 2. 月球錨點子系統

### 2.1 設計目標
- 固定纜索，承受約 12,000 kN 拉力
- 長期穩定於月球南極

### 2.2 技術方案

| 組件 | 規格 | 說明 |
|:---|:---|:---|
| 位置 | 月球南極（88.5°S） | 引用 Luna-Grid |
| 固定方式 | 鑽孔 + 月壤灌注 | 深度 100 米 |
| 材料 | 月壤固化 | 強度約 30 MPa |
| 連接器 | 鈦合金 | 抗腐蝕、高強度 |

### 2.3 建造流程

| 步驟 | 說明 | 時間 |
|:---|:---|:---|
| 1 | 鑽孔（100 米深） | 1 個月 |
| 2 | 安裝連接器 | 1 週 |
| 3 | 灌注液態月壤 | 1 週 |
| 4 | 冷卻固化 | 1 週 |

## 3. 升降機 (Climber) 子系統

### 3.1 設計目標
- 沿纜索爬升，運送 10 噸貨物
- 核聚變輔助，2 小時可達 L1 點

### 3.2 技術方案

| 組件 | 規格 | 說明 |
|:---|:---|:---|
| 驅動方式 | 電動滾輪 | 夾緊纜索 |
| 動力 | 太陽能 + 核聚變無線傳輸 | |
| 速度 | 100 km/h | 可調節 |
| 載重 | 10 噸 | |
| 制動系統 | 電磁制動 | 緊急停機 |

### 3.3 核聚變輔助（引用 Project Helios）

| 參數 | 數值 |
|:---|:---|
| 核聚變功率 | 500 MW |
| 無線傳輸效率 | 60% |
| 接收功率 | 300 MW |
| 爬升時間 | **2 小時** |

## 5. 微隕石防護子系統

### 5.1 設計目標
- 偵測並避免纜索被切斷
- 修復受損部位

### 5.2 技術方案

| 組件 | 規格 | 說明 |
|:---|:---|:---|
| 監測 | 雷達 + 光學感測 | 偵測 1 cm 以上碎片 |
| 清除 | 雷射清除系統 | 蒸發碎片 |
| 結構 | 多股編織 | 1,000 條幼線 |
| 修復 | 機械人 + 備用纖維 | 自動修復 |

### 5.3 風險評估

| 碎片大小 | 影響 | 應對 |
|:---|:---|:---|
| <1 cm | 無影響 | 無需處理 |
| 1-10 cm | 可能切斷數條幼線 | 雷射清除 |
| >10 cm | 可能切斷多條 | 雷射清除 + 機械人修復 |
